## GO

### 语言特点

1. C + Python 既有静态语言的安全性又有动态语言的开发效率

2. 有指针的概念 *

3. 有包的概念， go的每一个文件都属于一个包，不能单独存在

4. 天然并发 goroutine,轻量线程，可实现大并发处理，高效利用多核

5. 垃圾回收机制， 内存自动回收

6. 管道通信机制， goroutine通信

7. 可以返回多个值  return a, b

8. 切片， 延时执行函数defer 


### 指令

go run main.go  ----> 执行结果
go build main.go  ----> 可执行二进制文件

gofmt main.go  ---> 代码风格
